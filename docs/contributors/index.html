Contributor Guide

  1. Like, star and fork the repo if you want to help with visibility
  2. Read all of the docs on this site!
  3. Have a look at the issue board and choose something to work on
  4. Before starting any work, please at least say "Hi" on the giveth2-dev/Giveth2 construction office channel (see chat links here: https://giveth.io/join)
  5. Even better, after introducing yourself pls join the next DEV Call (usually Sunday 17:30 CEST, but watch announcements in the discord channel)
  6. If you found the project via a Gitcoin bounty, please make sure to check the issue every few days until completion (Gitcoin requires some proof of life checks)
  7. Please start a new branch on your fork named with the feature/fix you want to attempt and make a pull request when you are confident everything is ready to publish
